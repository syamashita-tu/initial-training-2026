1. 線形時不変システム

4. 畳み込み定理

巡回畳み込み

NNN 点周期の離散時間信号 xNx_NxN​ を 線形時不変システム H\mathcal{H}H に入力することを考える． このとき，出力 yyy は，

y[n]=∑k=0N−1xN[k]h[n−k]=k=l+mN∑l=0N−1∑m=−∞∞xN[l+mN]h[n−l−mN]=∑l=0N−1xN[l]∑m=−∞∞h[n−l−mN]y[n] = \sum_{k=0}^{N-1} x_N[k] h[n-k] \overset{k=l+mN}{=} \sum_{l=0}^{N-1} \sum_{m=-\infty}^{\infty} x_N[l+mN] h[n-l-mN] = \sum_{l=0}^{N-1} x_N[l] \sum_{m=-\infty}^{\infty} h[n-l-mN] y[n]=k=0∑N−1​xN​[k]h[n−k]=k=l+mNl=0∑N−1​m=−∞∑∞​xN​[l+mN]h[n−l−mN]=l=0∑N−1​xN​[l]m=−∞∑∞​h[n−l−mN]

ここで， hN[n]=∑m=−∞∞h[n−mN]h_N[n] = \sum_{m=-\infty}^{\infty} h[n-mN]hN​[n]=∑m=−∞∞​h[n−mN] と定義すると，出力 yyy は，

y[n]=∑l=0N−1xN[l]hN[n−l]=∑l=0N−1hN[l]xN[n−l]y[n] = \sum_{l=0}^{N-1} x_N[l] h_N[n-l] = \sum_{l=0}^{N-1} h_N[l] x_N[n-l] y[n]=l=0∑N−1​xN​[l]hN​[n−l]=l=0∑N−1​hN​[l]xN​[n−l]

と表せる． この演算は， xNx_NxN​ と hNh_NhN​ の NNN点 巡回畳み込み (circular convolution) と呼ばれ， xN⊛NhNx_N \circledast_N h_NxN​⊛N​hN​ と表される． また，非周期信号 hhh から NNN 点周期の信号 hN=∑m=−∞∞h[n−mN]h_N = \sum_{m=-\infty}^{\infty} h[n-mN]hN​=∑m=−∞∞​h[n−mN] を得る操作を， hhh の NNN 点 周期化 と呼ぶ．

巡回畳み込みは，線形畳み込みの結果 yyy をNNN点周期化したものと一致する．

信号の周期化は，次のように実装できる．

import numpy as np

def periodize(x, N):
    x = np.asarray(x)
    y = np.zeros(N, dtype=x.dtype)
    np.add.at(y, np.arange(x.size) % N, x) # nearly equivalent to y[np.arange(x.size) % N] += x
    return y

x = np.arange(6)  # [0, 1, 2, 3, 4, 5]
print(periodize(x, 4))

また，巡回畳み込みは， periodize 関数を用いて次のように実装できる．

import numpy as np

def circular_convolve_direct(x, h, N=None):
    x = np.asarray(x)
    h = np.asarray(h)

    if N is None:
        if x.size != h.size:
            raise ValueError("x と h の長さが異なる場合は N を指定してください。")
        N = x.size

    xN = periodize(x, N)
    hN = periodize(h, N)

    dtype = np.result_type(xN, hN, np.float64)
    y = np.zeros(N, dtype=dtype)

    for m in range(N):
        y += hN[m] * np.roll(xN, m)

    return y

x = np.array([1, 2, 3])
h = np.array([4, 5, 6])
y = circular_convolve_direct(x, h, N=3)

畳み込み定理

NNN 点の信号は周期 NNN の複素正弦波信号の線形結合で表せる（逆DFT）ことを利用して， システム H\mathcal{H}H の入出力関係を考えてみよう．

NNN 点の入力信号 xxx は，そのDFT XXX を用いて次のように表すことができる．

xˇ=1N∑k=0N−1X[k]ϕk\check{x} = \frac{1}{N} \sum_{k=0}^{N-1} X[k] \phi_k xˇ=N1​k=0∑N−1​X[k]ϕk​

y=H{xˇ}=H{1N∑k=0N−1X[k]ϕk}=1N∑k=0N−1X[k]H{ϕk}=1N∑k=0N−1X[k]HN[k]ϕky = \mathcal{H}\{\check{x}\} = \mathcal{H}\left\{\frac{1}{N} \sum_{k=0}^{N-1} X[k] \phi_k \right\} = \frac{1}{N} \sum_{k=0}^{N-1} X[k] \mathcal{H}\{\phi_k\} = \frac{1}{N} \sum_{k=0}^{N-1} X[k] H_N[k] \phi_k y=H{xˇ}=H{N1​k=0∑N−1​X[k]ϕk​}=N1​k=0∑N−1​X[k]H{ϕk​}=N1​k=0∑N−1​X[k]HN​[k]ϕk​

この式は， X[k]X[k]X[k] と HN[k]H_N[k]HN​[k] の積の逆DFTが出力 yyy になることを示している． すなわち， yyy のDFTは，

Y[k]=X[k]HN[k]Y[k] = X[k] H_N[k] Y[k]=X[k]HN​[k]

となる．
