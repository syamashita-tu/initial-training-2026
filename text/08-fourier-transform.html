解説

1. Fourier (フーリエ) 変換

信号を、（複素）正弦波の重ね合わせの形に分解することを フーリエ (Fourier) 変換 という。 Fourier 変換は、対象とする信号の種類によっていくつかの種類がある。 ここでは，コンピュータを用いた数値計算に適した 離散フーリエ変換 (discrete Fourier transform, DFT) を紹介する。

ベクトルの分解

任意の NNN 次元ベクトル x\boldsymbol{x}x は， NNN 個の線形独立なベクトル e1,⋯ ,eN\boldsymbol{e}_1, \cdots, \boldsymbol{e}_Ne1​,⋯,eN​ の和として 次式のとおり表せる．

x=a1e1+⋯+aNeN\boldsymbol{x} = a_1 \boldsymbol{e}_1 + \cdots + a_N \boldsymbol{e}_N x=a1​e1​+⋯+aN​eN​

ここで，a1,⋯ ,aNa_1, \cdots, a_Na1​,⋯,aN​ はスカラーである． このとき，e1,⋯ ,eN\boldsymbol{e}_1, \cdots, \boldsymbol{e}_Ne1​,⋯,eN​ を x\boldsymbol{x}x の 基底 (basis) という．

言い換えると， ベクトル x\boldsymbol{x}x は，より単純なベクトル e1,⋯ ,eN\boldsymbol{e}_1, \cdots, \boldsymbol{e}_Ne1​,⋯,eN​ に分解できる． このことは，複雑な対象（ベクトル）を解析するために役立つ． 例えば，平面上の運動 x=(x1,x2)⊤\boldsymbol{x} = (x_1, x_2)^\topx=(x1​,x2​)⊤ は， e1=(1,0)⊤\boldsymbol{e}_1 = (1, 0)^\tope1​=(1,0)⊤ と e2=(0,1)⊤\boldsymbol{e}_2 = (0, 1)^\tope2​=(0,1)⊤ を用いて 水平方向の運動と垂直方向の運動に分解して考えることができる．

x=x1e1+x2e2\boldsymbol{x} = x_1 \boldsymbol{e}_1 + x_2 \boldsymbol{e}_2 x=x1​e1​+x2​e2​

基底 e1,⋯ ,eN\boldsymbol{e}_1, \cdots, \boldsymbol{e}_Ne1​,⋯,eN​ が互いに直交しているとき， すなわち，

⟨ei,ej⟩={αi=j0i≠j\langle \boldsymbol{e}_i, \boldsymbol{e}_j \rangle = \begin{cases} \alpha & i = j \\ 0 & i \neq j \end{cases} ⟨ei​,ej​⟩={α0​i=ji=j​

を満たすとき，e1\boldsymbol{e}_1e1​ と x\boldsymbol{x}x の内積をとると，e1\boldsymbol{e}_1e1​ の係数 a1a_1a1​ を得られる．

⟨e1,x⟩=⟨e1,a1e1+⋯+aNeN⟩=a1⟨e1,e1⟩+⋯+aN⟨e1,eN⟩=a1α\langle \boldsymbol{e}_1, \boldsymbol{x} \rangle = \langle \boldsymbol{e}_1, a_1 \boldsymbol{e}_1 + \cdots + a_N \boldsymbol{e}_N \rangle = a_1 \langle \boldsymbol{e}_1, \boldsymbol{e}_1 \rangle + \cdots + a_N \langle \boldsymbol{e}_1, \boldsymbol{e}_N \rangle = a_1 \alpha ⟨e1​,x⟩=⟨e1​,a1​e1​+⋯+aN​eN​⟩=a1​⟨e1​,e1​⟩+⋯+aN​⟨e1​,eN​⟩=a1​α

よって，a1=⟨e1,x⟩/αa_1 = \langle \boldsymbol{e}_1, \boldsymbol{x} \rangle / \alphaa1​=⟨e1​,x⟩/α となる． 特に，α=1\alpha = 1α=1 のとき，a1=⟨e1,x⟩a_1 = \langle \boldsymbol{e}_1, \boldsymbol{x} \ranglea1​=⟨e1​,x⟩ となる．

複素正弦波信号

複素数値離散時間信号 ejωne^{j \omega n}ejωn を，角周波数 ω\omegaω の離散時間 複素正弦波信号 という． オイラーの公式 ejθ=cos⁡(θ)+jsin⁡(θ)e^{j \theta} = \cos(\theta) + j \sin(\theta)ejθ=cos(θ)+jsin(θ) を使うと，複素正弦波信号は次のように表せる．

ejωn=cos⁡(ωn)+jsin⁡(ωn)e^{j \omega n} = \cos(\omega n) + j \sin(\omega n) ejωn=cos(ωn)+jsin(ωn)

すなわち，複素正弦波信号は，実部が角周波数 ω\omegaω の余弦波，虚部が角周波数 ω\omegaω の正弦波である． 複素正弦波信号は 2π/ω2 \pi / \omega2π/ω が有理数となる場合に限り周期信号となり，その時の周期は N=2π/ωN = 2 \pi / \omegaN=2π/ω である．

長さが NNN であり，かつ，周期 NNN を持つ複素正弦波信号は，次の式で表される．

ϕk[n]=ej2πkn/N,k,n∈{0,1,⋯ ,N−1}\phi_k[n] = e^{j 2 \pi k n / N}, \quad k, n \in \{0, 1, \cdots, N-1\} ϕk​[n]=ej2πkn/N,k,n∈{0,1,⋯,N−1}

となる．

離散フーリエ変換 (discrete Fourier transform, DFT)

複素正弦波信号 ϕ0,⋯ ,ϕN−1\phi_0, \cdots, \phi_{N-1}ϕ0​,⋯,ϕN−1​ は線形独立であるため，これらを基底として 離散時間信号 x[n]x[n]x[n] を次のように分解できる．

x[n]=1N(X[0]ϕ0[n]+⋯+X[N−1]ϕN−1[n])x[n] = \frac{1}{N} \left( X[0] \phi_0[n] + \cdots + X[N-1] \phi_{N-1}[n] \right) x[n]=N1​(X[0]ϕ0​[n]+⋯+X[N−1]ϕN−1​[n])

X[k]=⟨ϕk,x⟩X[k] = \langle \phi_k, x \rangle X[k]=⟨ϕk​,x⟩

と表される． また，1N\frac{1}{N}N1​ は， ⟨ϕk,ϕk⟩=N\langle \phi_k, \phi_k \rangle = N⟨ϕk​,ϕk​⟩=N であることを考慮した正規化係数である．

このとき， x[n]x[n]x[n] から X[k]X[k]X[k] への変換を 離散フーリエ変換 (discrete Fourier transform, DFT) という．

X=FNx\boldsymbol{X} = \boldsymbol{F}_N \boldsymbol{x} X=FN​x

と表される． この行列 FN\boldsymbol{F}_NFN​ を NNN 点の DFT 行列 という．

また，X[k]X[k]X[k] から x[n]x[n]x[n] への変換を 逆離散フーリエ変換 (inverse discrete Fourier transform, IDFT) という

振幅スペクトルと位相スペクトル

DFTの結果 X[k]X[k]X[k] は複素数となる． このとき，X[k]X[k]X[k] の大きさ ∣X[k]∣=Re(X[k])2+Im(X[k])2|X[k]| = \sqrt{\mathrm{Re}(X[k])^2 + \mathrm{Im}(X[k])^2}∣X[k]∣=Re(X[k])2+Im(X[k])2​ を 振幅スペクトル (amplitude spectrum) といい， X[k]X[k]X[k] の偏角 ∠X[k]=atan2(Im(X[k]),Re(X[k]))\angle X[k] = \mathrm{atan2}(\mathrm{Im}(X[k]), \mathrm{Re}(X[k]))∠X[k]=atan2(Im(X[k]),Re(X[k])) を 位相スペクトル (phase spectrum) という．

信号 x[n],x1[n],x2[n]x[n], x_1[n], x_2[n]x[n],x1​[n],x2​[n] の NNN 点 DFT をそれぞれ X[k],X1[k],X2[k]X[k], X_1[k], X_2[k]X[k],X1​[k],X2​[k] とすると， DFT は次の性質を満たす．

3. 高速フーリエ変換 (fast Fourier transform, FFT)

DFTを定義に従い直接計算すると，NNN 点の信号に対して O(N2)O(N^2)O(N2) の計算量が必要となる． しかし，計算を工夫することで，O(Nlog⁡N)O(N \log N)O(NlogN) の計算量で同じ結果を得ることができる． このようにDFTを高速に計算するアルゴリズムの総称を 高速フーリエ変換 (fast Fourier transform, FFT) という． FFTは，DFTを計算するためのアルゴリズムであるため，FFTを使って得られる結果はDFTと同じである．
