解説

1. 信号とは

信号 (signal) とは，情報を伝達する音や光などに対するその物理量． 信号は，1つ以上の独立変数を有する関数・数列・ベクトルとして表される． 以下では，時間を独立変数とする信号を考える．

連続時間信号

任意の時間 t∈Rt \in \mathbb{R}t∈R に対して定義される信号で，x(t)x(t)x(t) と表す（丸括弧）．

離散時間信号

とびとびの時間（整数 n∈Zn \in \mathbb{Z}n∈Z） に対して定義される信号で，x[n]x[n]x[n] と表す（角括弧）．

x[n]x[n]x[n] を要素とするベクトル x=(⋯ ,x[n−1],x[n],x[n+1],⋯ )⊤ \boldsymbol{x} = (\cdots, x[n-1], x[n], x[n+1], \cdots)^\topx=(⋯,x[n−1],x[n],x[n+1],⋯)⊤ として表現する場合もある．

正弦波信号と周期信号

実用上重要な信号の多くは、正弦波の重ね合わせ（和）の形に分解できる（Fourier変換）． このため，信号処理においては正弦波が非常に重要である．

連続時間の正弦波信号は次の式で表される．

x(t)=Asin⁡(2πFt+ϕ)=Asin⁡(Ωt+ϕ)x(t) = A \sin(2\pi F t + \phi) = A \sin(\Omega t + \phi) x(t)=Asin(2πFt+ϕ)=Asin(Ωt+ϕ)

  • AAA：振幅 (Amplitude)
  • FFF: 周波数 (frequency) [Hz]
  • T=1/FT = 1 / FT=1/F: 周期 (period) [s]
  • ϕ\phiϕ: 位相 (phase)
  • Ω=2πF\Omega = 2\pi FΩ=2πF：角周波数 (angular frequency) [rad/s]

正弦波信号のように， x(t)=x(x+T)x(t) = x(t+T)x(t)=x(x+T) を満たす TTT が存在する信号を 周期信号 という．

標本化（サンプリング）

連続時間信号 x(t)x(t)x(t) を一定間隔 TsT_sTs​ で取り出し、離散時間信号 x[n]x[n]x[n] を得る操作を 標本化 (sampling) という．

x[n]=x(nTs),n=0,1,2,…x[n] = x(n T_s), \quad n = 0, 1, 2, \ldots x[n]=x(nTs​),n=0,1,2,…

  • TsT_sTs​：サンプリング周期 (sampling period) [s]
  • Fs=1/TsF_s = 1 / T_sFs​=1/Ts​：サンプリング周波数 (sampling frequency) [Hz]

Python での生成例

import numpy as np
import matplotlib.pyplot as plt
 
Fs = 1000             # サンプリング周波数 [Hz]
Ts = 1.0 / fs         # サンプリング周期 [秒]
duration = 1.0        # 信号長 [秒]
 
n = np.arange(int(duration * Fs))     # 0, 1, 2, ..., N-1
t = n * Ts                            # 各サンプルに対応する時刻
 
F = 5.0
x = np.sin(2 * np.pi * F * t)        # x[n] = sin(2π f0 (n Ts))
 
plt.plot(t, x)
plt.xlabel("Time [s]"); plt.ylabel("x[n]")
plt.show()

2. サンプリング定理

ある条件を満たせば，離散時間信号 x[n]x[n]x[n] からもともとの連続時間信号 x(t)x(t)x(t) を完全に復元できる． この条件を述べる定理がサンプリング定理である．

サンプリング定理（ナイキスト・シャノンの定理）

Fmax⁡F_{\max}Fmax​ [Hz] 未満の周波数成分のみを含む（= Fmax⁡F_{\max}Fmax​ [Hz] 未満の周波数を持つ正弦波の和で表せる）連続時間信号を考える． このとき，

Fs>2Fmax⁡F_s > 2 F_{\max} Fs​>2Fmax​

を満たすサンプリング周波数 FsF_sFs​ で信号を標本化すれば、その離散時間信号から元の連続時間信号を完全に復元できる。

  • FNyq=Fs/2F_{\mathrm{Nyq}} = F_s / 2FNyq​=Fs​/2: ナイキスト周波数 (Nyquist frequency)

エイリアシング

Fmax⁡>FNyqF_{\max} > F_{\mathrm{Nyq}}Fmax​>FNyq​ の場合， FNyqF_{\mathrm{Nyq}}FNyq​ を超える高周波の成分が 偽の低周波成分として観測される 現象が起きる．これを エイリアシング (aliasing) または 折り返し と呼ぶ．

3. 信号に対する演算

連続時間信号 x(t)x(t)x(t) や離散時間信号 x[n]x[n]x[n] に対して，以下のような基本演算が定義される．

時間シフト

信号を平行移動（遅延）させる

y(t)=x(t−τ),y[n]=x[n−k]y(t) = x(t - \tau), \quad y[n] = x[n - k] y(t)=x(t−τ),y[n]=x[n−k]

  • τ∈R,k∈Z\tau \in \mathbb{R}, k \in \mathbb{Z}τ∈R,k∈Z：シフト量

時間反転

信号を時間軸に対して反転させる．

y(t)=x(−t),y[n]=x[−n]y(t) = x(-t), \quad y[n] = x[-n] y(t)=x(−t),y[n]=x[−n]

時間伸縮

信号を時間軸に関して伸び縮みさせる．

y(t)=x(at),y[n]=x[an]y(t) = x(at), \quad y[n] = x[an] y(t)=x(at),y[n]=x[an]

  • a>0a > 0a>0：伸縮倍率
  • ただし，ananan が整数でない場合は x[an]=0x[an] = 0x[an]=0 とする

振幅スケーリング・オフセット

y(t)=a⋅x(t)+by(t) = a \cdot x(t) + b y(t)=a⋅x(t)+b

  • aaa：振幅の倍率（負の数なら符号反転）
  • bbb：オフセット（振幅方向の平行移動量）

信号の加算・乗算

2つの信号 x1,x2x_1, x_2x1​,x2​ に対して：

  • 加算：y(t)=x1(t)+x2(t),y[n]=x1[n]+x2[n]y(t) = x_1(t) + x_2(t), \quad y[n] = x_1[n] + x_2[n]y(t)=x1​(t)+x2​(t),y[n]=x1​[n]+x2​[n]
  • 乗算：y(t)=x1(t)⋅x2(t),y[n]=x1[n]⋅x2[n]y(t) = x_1(t) \cdot x_2(t), \quad y[n] = x_1[n] \cdot x_2[n]y(t)=x1​(t)⋅x2​(t),y[n]=x1​[n]⋅x2​[n]
  • 内積：⟨x1,x2⟩=∫Rx1(t)x2(t)dt,⟨x1,x2⟩=∑−∞∞x1[n]x2[n]=x1⊤x2\langle x_1, x_2 \rangle = \int_\mathbb{R} x_1(t) x_2(t) \mathrm{d}t, \quad \langle x_1, x_2 \rangle = \sum_{-\infty}^{\infty} x_1[n] x_2[n] = \boldsymbol{x}_1^\top \boldsymbol{x}_2⟨x1​,x2​⟩=∫R​x1​(t)x2​(t)dt,⟨x1​,x2​⟩=∑−∞∞​x1​[n]x2​[n]=x1⊤​x2​

4. 量子化

コンピュータでは，時間も振幅も離散的なディジタル信号しか扱うことができない． 連続的な振幅値を有限個の代表値に対応させることを 量子化 (quantization) と呼ぶ．

量子化の基本

  • ビット深度 BBB：代表値を表現するために用いるビット数
  • 量子化レベル数 L=2BL = 2^BL=2B：代表値の個数
  • 量子化間隔 Δ\DeltaΔ：代表値間の間隔．量子化間隔が一定である量子化を一様量子化という
  • もとの振幅値 xxx と量子化後の振幅値 x^\hat{x}x^ の間の誤差 e=x^−xe = \hat{x} - xe=x^−x を 量子化誤差 という

Python での量子化

def quantize(x, n_bits, x_min=-1.0, x_max=1.0):
    """振幅を [x_min, x_max] の範囲で n_bits 段階に量子化"""
    L = 2 ** n_bits
    step = (x_max - x_min) / (L - 1)
    return np.round((x - x_min) / step) * step + x_min
