Pythonを用いた微分・線形代数・確率の演習問題一覧

1. 基礎：関数の値とグラフを調べる

関数

\[ f(x)=x^2 \]

について、次のことを行え。

手計算

以下の値を手計算せよ。

\[ f(-2),\ f(-1),\ f(0),\ f(1),\ f(2) \]

また、計算結果からグラフがどのような形になりそうか説明せよ。

Python 課題

numpy を用いて、\(-3 \leq x \leq 3\) の範囲で \(f(x)=x^2\) を計算し、グラフを描け。

検証・考察

手計算で求めた点が、Python で描いたグラフ上にあることを確認せよ。また、\(x=0\) の近くでグラフがどのように曲がっているか説明せよ。

2. 基礎：変化量から「傾き」を理解する

関数

\[ f(x)=x^2 \]

について、\(x=2\) の近くでの傾きを調べる。

手計算

次の値を計算せよ。

\[ \frac{f(2.1)-f(2)}{2.1-2},\quad \frac{f(2.01)-f(2)}{2.01-2},\quad \frac{f(2.001)-f(2)}{2.001-2} \]

これらの値がどの数に近づいているか考えよ。

Python 課題

差分幅 \(h\) を小さくしながら、

\[ \frac{f(2+h)-f(2)}{h} \]

を計算せよ。

検証・考察

\(h\) を小さくすると、傾きがどの値に近づくか確認せよ。理論的には、

\[ f'(x)=2x \]

なので、\(x=2\) での傾きは \(4\) である。数値計算の結果と一致するか確認せよ。

3. 基礎：接線をグラフで確認する

関数

\[ f(x)=x^2 \]

の \(x=1\) における接線を求め、グラフに重ねて描け。

手計算

まず、

\[ f(1) \]

を求めよ。次に、導関数

\[ f'(x)=2x \]

を用いて、\(x=1\) における傾きを求めよ。

接線の式は、

\[ y=f(1)+f'(1)(x-1) \]

で与えられる。この式を整理せよ。

Python 課題

関数 \(f(x)=x^2\) と、求めた接線を同じグラフに描け。

検証・考察

接線が \(x=1\) の点でグラフに接していることを確認せよ。接線が「その点の近くで関数を直線で近似している」と言える理由を説明せよ。

4. 発展：勾配降下法で最小値を探す

関数

\[ f(x)=x^2-4x+5 \]

の最小値を、手計算と Python の両方で求めよ。

手計算

平方完成により、

\[ f(x)=x^2-4x+5 \]

を

\[ f(x)=(x-a)^2+b \]

の形に変形せよ。その結果から、最小値をとる \(x\) と最小値を求めよ。

また、導関数

\[ f'(x)=2x-4 \]

を用いて、

\[ f'(x)=0 \]

を解き、同じ結果が得られることを確認せよ。

Python 課題

初期値 \(x=6\) から出発し、次の更新式で最小値に近づく様子を確認せよ。

\[ x_{\text{new}}=x-\eta f'(x) \]

ここで、\(\eta\) は学習率と呼ばれる小さな正の値である。

検証・考察

反復を重ねると \(x\) がどの値に近づくか確認せよ。また、学習率 \(\eta\) を 0.01, 0.1, 0.8, 1.1 などに変えたとき、収束の様子がどのように変わるか観察せよ。

5. 基礎：ベクトルの長さと内積

次の 2 つのベクトルを考える。

\[ a= \begin{pmatrix} 3\\ 4 \end{pmatrix}, \quad b= \begin{pmatrix} 1\\ 2 \end{pmatrix} \]

手計算

次を求めよ。

\[ \|a\|,\quad \|b\|,\quad a\cdot b \]

ここで、\(\|a\|\) はベクトル \(a\) の長さ、\(a\cdot b\) は内積である。

Python 課題

numpy を使って、同じ計算を行え。さらに、2 つのベクトルを矢印として図示せよ。

検証・考察

ベクトル \(a\) の長さが \(5\) になることを、三平方の定理と対応づけて説明せよ。また、内積が大きいとき、2 つのベクトルの向きにはどのような関係がありそうか考えよ。

6. 基礎：行列によるベクトルの変換

次の行列を考える。

\[ A= \begin{pmatrix} 2 & 0\\ 0 & 1 \end{pmatrix} \]

この行列をベクトル

\[ v= \begin{pmatrix} 1\\ 2 \end{pmatrix} \]

に作用させた結果 \(Av\) を求めよ。

手計算

行列とベクトルの積を計算せよ。

\[ Av= \begin{pmatrix} 2 & 0\\ 0 & 1 \end{pmatrix} \begin{pmatrix} 1\\ 2 \end{pmatrix} \]

Python 課題

numpy で同じ計算を行い、変換前後のベクトルを描画せよ。

検証・考察

この行列は、ベクトルのどの成分を変化させているか説明せよ。特に、\(x\) 方向と \(y\) 方向の変化に注目せよ。

7. 発展：回転行列で点を回す

点

\[ v= \begin{pmatrix} 1\\ 0 \end{pmatrix} \]

を原点中心に \(90^\circ\) 回転させる。

回転行列は次で与えられる。

\[ R= \begin{pmatrix} \cos\theta & -\sin\theta\\ \sin\theta & \cos\theta \end{pmatrix} \]

手計算

\(\theta=90^\circ\) のとき、

\[ \cos 90^\circ=0,\quad \sin 90^\circ=1 \]

である。したがって、回転行列 \(R\) を具体的に書き、\(Rv\) を計算せよ。

Python 課題

角度を \(0^\circ, 30^\circ, 60^\circ, 90^\circ, 180^\circ\) と変えながら、点 \((1,0)\) がどこに移動するか描画せよ。

検証・考察

回転後もベクトルの長さが \(1\) のままであることを確認せよ。回転行列は「長さを変えずに向きだけを変える変換」である、という説明が妥当か検討せよ。

8. 発展：固有値と固有ベクトルを可視化する

次の行列を考える。

\[ A= \begin{pmatrix} 2 & 0\\ 0 & 3 \end{pmatrix} \]

この行列により、いくつかのベクトルがどのように変換されるか調べよ。

手計算

次の 2 つのベクトルについて、\(Av\) を計算せよ。

\[ e_1= \begin{pmatrix} 1\\ 0 \end{pmatrix}, \quad e_2= \begin{pmatrix} 0\\ 1 \end{pmatrix} \]

それぞれ、向きが変わるかどうか確認せよ。

Python 課題

np.linalg.eig を用いて、固有値と固有ベクトルを求めよ。さらに、ランダムなベクトルと固有ベクトルを変換前後で描画し、向きの変化を比較せよ。

検証・考察

\((1,0)\) と \((0,1)\) は、行列をかけても向きが変わらないことを確認せよ。一方で、\((1,1)\) は一般には向きが変わることを説明せよ。

9. 基礎：乱数で確率を理解する

公平なサイコロを 1 回振ったとき、偶数が出る確率を考える。

手計算

サイコロの出目は

\[ 1,2,3,4,5,6 \]

である。このうち偶数は

\[ 2,4,6 \]

である。したがって、偶数が出る確率を求めよ。

Python 課題

サイコロを \(10\), \(100\), \(1000\), \(10000\) 回振るシミュレーションを行い、偶数が出た割合を計算せよ。

また、試行回数を増やしたとき、偶数が出る割合がどのように変化するかグラフにせよ。

検証・考察

試行回数が少ないと理論値からずれるが、試行回数が多くなると \(0.5\) に近づくことを確認せよ。この性質を「大数の法則」と呼ぶ。

10. 発展：正規分布と平均・分散をシミュレーションする

平均 \(0\)、標準偏差 \(1\) の正規分布に従う乱数を多数生成し、平均と分散を調べよ。

手計算

次の小さなデータについて、平均と分散を手計算せよ。

\[ x = [1,\ 2,\ 3,\ 4,\ 5] \]

平均は

\[ \bar{x}=\frac{x_1+x_2+\cdots+x_n}{n} \]

である。分散は、各データが平均からどれくらい離れているかを表す量であり、ここでは

\[ \frac{1}{n}\sum_{i=1}^{n}(x_i-\bar{x})^2 \]

として計算せよ。

Python 課題

まず、手計算したデータについて、Python で平均と分散を確認せよ。

次に、平均 \(0\)、標準偏差 \(1\) の正規分布に従う乱数を生成し、平均、分散、ヒストグラムを確認せよ。

検証・考察

生成した乱数の平均が \(0\) に近く、分散が \(1\) に近いことを確認せよ。ただし、乱数なので完全には一致しない。そのずれが、データ数を増やすと小さくなるかどうか調べよ。
